# 铁柱工具箱PRO

一款面向《第七史诗》的 Windows 装备强化辅助工具。通过 ADB 连接安卓模拟器并截取游戏画面，使用本地 OCR 自动识别装备信息、计算民间分数、给出强化建议，并按人工维护的套装属性需求推荐装备用途。

[下载最新版](https://github.com/Mooooooon/TiezhuToolboxPRO/releases/latest)

## 主要功能

- 自动识别装备等级、强化等级、品质、主属性、副属性和套装
- 根据副属性计算民间分数，并按装备等级和强化阶段给出强化建议
- 支持按强化建议自动强化装备，并按设置保留、出售或分解装备
- 按“套装 → 属性子类 → 英雄配装组合”推荐装备用途，支持不同流派使用独立属性权重
- 右三主属性按85→90预估及88/90满值量化，速度鞋、暴击项链等自然参与需求匹配
- 内置只读需求分析页，可查看每个套装的属性组合、权重和适用英雄配装
- 支持速度套必须带速度、暴击率/暴伤高权重项链仅接受对应主属性等可关闭的强化规则
- 支持自定义强化阈值、最低需求匹配度、单次自动处理上限、识别快捷键和持续识别间隔
- 内置精简版 ADB，无需额外安装 .NET 运行环境

## 界面预览

### 装备强化

连接模拟器后，可一键截图识别装备，并查看分数、强化建议和适用角色。

![装备强化界面](docs/images/equipment-enhancement.png)

### 自动强化

在游戏中打开英雄装备列表并选中第一件装备后，程序会通过截图确认当前界面和按钮位置，自动识别装备、执行分阶段强化，并依据强化建议保留或淘汰装备。可设置淘汰时出售或分解、单次处理上限，以及遇到符合保留条件的装备后停止或继续下一件。

> 自动强化会消耗游戏资源，并可能出售或分解装备。启动前请核对强化阈值、处理方式和单次上限。

![自动强化界面](docs/images/automatic-enhancement.png)

### 需求分析

按套装查看人工维护的属性子类、聚合权重、需求量及英雄完整配装组合。

### 软件设置

可调整装备分数阈值、自动强化处理方式与上限、最低需求匹配度、速度套/暴击项链特殊规则、全局识别快捷键和持续识别间隔。

![软件设置界面](docs/images/app-settings.png)

## 使用方法

1. 从 [Releases](https://github.com/Mooooooon/TiezhuToolboxPRO/releases/latest) 下载最新版 ZIP 并完整解压。
2. 在安卓模拟器中开启 ADB 调试。MuMu 12 的默认地址为 `127.0.0.1:16384`。
3. 启动 `铁柱工具箱PRO.exe`，选择已发现的设备；如果没有发现设备，输入 ADB 地址后点击“连接”。
4. 在游戏中进入需要分析的装备的强化页面，然后点击“截图识别”或按默认快捷键 `F2`。
5. 在“装备强化”页查看装备分数、强化建议、适用属性子类及英雄配装。
6. 如需自动强化，先在“软件设置”中确认处理方式和上限，再进入游戏英雄装备列表、选中第一件装备，然后在“自动强化”页启动任务。

## 使用要求

- Windows 10/11 64 位系统
- 支持 ADB 的安卓设备或模拟器
- 游戏使用简体中文界面

## 识别说明

- OCR 在本机运行，识别效果会受到游戏分辨率、界面缩放和截图清晰度影响。
- 当前强化建议主要针对 85 级和 88 级装备，并分别使用独立阈值。
- 需求推荐只匹配装备当前套装；匹配度同时考虑目标属性覆盖和分数向高权重属性的分配质量。
- 85级右三按未来90级满值估算，88/90使用同一满值档；主属性不加入装备分数和强化阶梯。
- 推荐结果仅供配装和强化决策参考，具体取舍仍应结合账号阵容。

## 装备强化判断免责声明

- 本工具提供的装备分数、强化建议和需求推荐基于民间算法、人工维护数据及用户设置的阈值，并非《第七史诗》官方结论，仅供参考。
- OCR 识别结果可能受到游戏版本、分辨率、界面缩放和画面质量等因素影响，请在强化前自行核对装备数据。
- 装备强化结果具有随机性，本工具无法保证实际强化结果或装备收益。因使用本工具建议而产生的游戏资源消耗、装备损失或其他后果，由使用者自行承担。
